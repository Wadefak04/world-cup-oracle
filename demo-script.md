# World Cup Oracle — Demo Video Script (3 minutes)

## Setup (before recording)
- Open `index.html` in browser (or testnet site via local portal)
- Have wallet ready in browser for demo if needed

---

## Scene 1: Introduction (0:00 - 0:30)
**Visual:** App homepage showing World Cup Oracle
**Narration:**
> "This is World Cup Oracle — an AI agent with persistent memory built on Walrus Protocol for the Walrus Memory World Cup hackathon. It tracks your FIFA World Cup 2026 predictions, learns your biases, and roasts you for them."

**Show:** Header, badge, wallet status (demo mode)

---

## Scene 2: Make Predictions (0:30 - 1:15)
**Visual:** Predict tab
**Actions:**
1. Click "Predict" tab
2. Select match: "🇧🇷 Brazil vs 🏴󠁧󠁢󠁥󠁮󠁧󠁿 England"
3. Pick: "🇧🇷 Brazil Win"
4. Confidence: 9/10
5. Reasoning: "Brazil always wins against European teams in group stage. Historical data supports this."
6. Click "Submit Prediction"
7. Show toast: "Prediction recorded! The Oracle remembers..."

**Narration:**
> "Every prediction you make — the pick, confidence level, and your reasoning — becomes permanent memory stored on Walrus."

---

## Scene 3: Memory Timeline (1:15 - 1:45)
**Visual:** Memory tab
**Actions:**
1. Click "Memory" tab
2. Show the timeline with the new prediction memory
3. Point out: Session 1, type: prediction, type: reasoning, type: trait

**Narration:**
> "The Oracle builds a memory timeline instantly. It remembers what you picked, why you picked it, and detects traits — like high confidence or team bias."

---

## Scene 4: Session Simulation — Day 1 vs Day 4+ (1:45 - 2:15)
**Visual:** Memory tab
**Actions:**
1. Click "New Session (Simulate Day N+1)"
2. Show Session 2 appears with "The Oracle has X predictions from previous sessions"
3. Make another prediction for different match
4. Go to Memory tab — show Session 2 memories

**Narration:**
> "This is the key innovation — persistent memory across sessions. On Day 1, the Oracle knows nothing. By Day 4, it references your history, detects patterns, and changes its behavior."

---

## Scene 5: Prediction Roast (2:15 - 2:45)
**Visual:** Roast tab
**Actions:**
1. Click "Roast" tab
2. Click "Roast My Predictions"
3. Read the generated roast aloud
4. Show it appears in memory as "roast" type

**Narration:**
> "The Oracle doesn't just remember — it analyzes. It detects overconfidence, team bias, draw addiction, and writes a personalized roast. This is genuine behavior change from memory."
